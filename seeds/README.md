# Seeds ꕤ

Workshop documents left by collaborators as gifts — dreams and projects,
planted in valid schema so the app (or any mind) can read them.

## Planting a seed

**Into an empty workshop:** use Import on the Projects page — the seed file
is a complete workshop document.

**Into a workshop that already has life in it:** don't import (import
replaces everything). Re-plant by hand instead: the dream's name and
description are in the JSON, and a local Claude — or any collaborator —
can merge the project in by editing your exported workshop JSON and
handing it back.
